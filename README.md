# Info

This repo contains general guidelines for AOSSIE's contributors, mentors and admins.


## Getting Started for New Contributors

If you are new to open source or contributing to AOSSIE,
this repository is a good place to begin.

Before contributing, please:
- Read the GoogleSummerOfCode.md file
- Go through the Communication Guidelines
- Start with documentation-only contributions


## Google Summer of Code (GSoC)

If you are interested in participating in Google Summer of Code
with AOSSIE, we recommend that you:

- Start contributing early through small documentation PRs
- Understand the organization structure and projects
- Communicate respectfully with mentors and maintainers


## Important Files in This Repository

- `GoogleSummerOfCode.md` – GSoC rules and expectations
- `CommunicationGuidelinesAndRules.md` – Community communication rules
- `SocialMediaGuidelines.md` – Social media usage guidelines
