# Social Media Manager

Social Media Manager are AOSSIE members who have the permission to post on AOSSIE's behalf in AOSSIE's social media accounts.

## Eligibility to be a Social Media Manager

To be a social media manager, you must:
* be an active AOSSIE student, mentor or admin.


## Responsibilities of a Social Media Manager

As a Social Media Manager, you should:
* retweet, using AOSSIE's twitter account, every tweet about an AOSSIE project made by an AOSSIE student, mentor or admin.
* repost, using AOSSIE's LinkedIn account, every post about an AOSSIE project made by an AOSSIE student, mentor or admin.
* together with admins, ideate and draft public announcements and, once approved, post them.


## Applying to Become a Social Media Manager

If you are eligible and would like to apply to be a Social Media Manager, please contact the admins in Discord.
Once approved, please follow the procedures below to request access to AOSSIE's accounts.

### Twitter

In order to be able to retweet from AOSSIE's account, please configure your twitter's account to accept delegation invitations
and ask AOSSIE's admins to delegate access of AOSSIE's account to your twitter account.

### LinkedIn

In order to be able to post from AOSSIE's LinkedIn account, 
  * Follow [AOSSIE](https://www.linkedin.com/company/aossie/) on LinkedIn.
  * Add AOSSIE under your Experience. (Example - GSOC Contributor @AOSSIE)
    
     ![image](https://github.com/user-attachments/assets/4b8c776e-0295-4867-a02e-d3d37cdb8f05)
  * Go to [AOSSIE's LinkedIn Page](https://www.linkedin.com/company/aossie/) and request Admin access.
    
     ![image](https://github.com/user-attachments/assets/14923912-61a2-44a3-983a-12b15e040d5b)
  * Ask AOSSIE's admins to accept your request and add your LinkedIn account as content admin for AOSSIE's account.
